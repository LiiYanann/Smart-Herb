# U.S. Consumer Digital Quality Passport v3 Warm

使用方式：
1. 把你的 Smarty IP 图片放到 `assets/images/1.png`。
2. 打开 `index.html`。
3. 右下角下载按钮会下载 `assets/reports/saffron-us-consumer-quality-passport.pdf`。

本版修改：
- 整体色调改为暖绿色/奶油色，减少阴森感。
- 左侧数字人区域补齐布局，不再左下空缺。
- 删除容易误解的 “Scan to...” 框，改为 U.S. Market Verified 状态卡。
- PDF 报告改为单页 A4，内容更完整，字体与排版优化。
- 语音使用浏览器可用的优先英文自然声线，并降低语速、微调音调。
